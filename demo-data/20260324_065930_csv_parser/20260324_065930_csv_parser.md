# CSV Parser

```python
import csv

def parse(file):
    totals = {}
    with open(file) as f:
        for row in csv.DictReader(f):
            cat = row["category"]
            totals[cat] = totals.get(cat, 0) + float(row["amount"])
    return sorted(totals.items(), key=lambda x: -x[1])
```
